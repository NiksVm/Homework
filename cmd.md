#Подсказки по командной строке

Команда смены директории
```sh
cd c:\folder_name
```

Команда отображения текущей директории(каталога)для macOS и Linux
```sh
pwd
```

Листинг текущей директории
```sh
dir
```
Linux, macOS:
```sh
```

Удаление файла которые не нужен
```sh
del <filename>
```
Linux, macOS
```sh
rm <filename>
```

Просмотреть логи
```sh
log
```
